The Workout Tracker Mobile Application (WTMA) is a fitness app designed to help users track their workouts. It allows users to create workout routines, log exercises, and monitor progress over time. Built using .NET MAUI, the app provides user-friendly interface, making it easy to add, edit, and organize workout sessions. With features like exercise selection, workout history, and a built-in timer.
